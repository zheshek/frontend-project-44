Brain Games — это набор консольных математических игр.
Игроку предлагается ответить на несколько вопросов подряд. Если все ответы правильные — игрок побеждает.

Проект реализован на Node.js и запускается через CLI (командную строку).

🚀 Возможности

🎮 Набор из нескольких логических игр
🧠 Проверка математических знаний
💻 Работа через CLI (терминал)
⚡ Быстрая и простая архитектура
🔁 Несколько раундов игры
📦 Простая установка через npm

🎯 Игры

Проект включает несколько игр:

🔢 Brain Even

Проверка числа на чётность.

Answer "yes" if the number is even, otherwise answer "no".
➕ Brain Calc

Вычисление результата арифметического выражения.

Question: 8 * 7
Your answer: 56
📊 Brain GCD

Поиск наибольшего общего делителя двух чисел.

Find the greatest common divisor of given numbers.
📈 Brain Progression

Определение пропущенного числа в арифметической прогрессии.

5 10 15 .. 25
🔍 Brain Prime

Определение простого числа.

Answer "yes" if given number is prime. Otherwise answer "no".
